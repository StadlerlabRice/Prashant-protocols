**Reco Mix (20x)**

|   |   |
|---|---|
|1M Glucose|2 ml|
|1M MgCl2|1 ml|
|1M MgSO4|1 ml|
|1M KCl|250 ul|
|Water Autocl|750 ul|
 
- 1:5 dilution of all components
- Autoclave all reagents separately and mix
- If using 20% glucose add : 1.8 ml
- LB-miller has 20 times the NaCl indicated in SOC so this recipe is **not accurate**
